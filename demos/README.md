# Tension Demos

This directory contains example projects and demos showcasing the Tension engine capabilities.

## Planned Demos

- **basic-boxes**: Simple moving rectangles demo
- **physics-simulation**: Physics-based interactions
- **input-handling**: Keyboard and mouse input examples
- **graphics-showcase**: Advanced rendering features
- **hot-reload-demo**: Demonstrating live code changes

## Creating a Demo

1. Create a new directory: `demos/your-demo/`
2. Add your WASM module source code
3. Include a build script and README
4. Test with the Tension core engine

Each demo should be self-contained and demonstrate specific engine features.
